## Getting Started

Para un mejor funcionamiento al momento de utilizar esta apps, debes instalarle paquete de dependencia de webpack webpack-cli y html-webpack-plugin

### Prerequisites

Instalación del visual studio code, node.js y git.

### Installation

Después de tener instaladas las herramientas mencionadas en los pre requisitos, abrir con visual studio code la carpeta donde se encuentran los archivos del aplicativo Todo. abrimos la terminal de VSC y ejecutamos el comando git init, git add . --- git commit, esto nos sirve para poder guardar versionados de nuestro código .. esto es posible si tienes instalados git de lo contrario realiza ese proceso. 
Para garantizar el funcionamiento instalamos los paquetes de dependencia de webpack y webpack-cli
con el siguiente codigo: npm install webpack webpack-cli
